MOLAND-Light

 

MOLAND Light is a simplified version of the MOLAND land use modelling software. It can be used to easily and quickly set up a simulation for a region in Europe, configure a core set of drivers of land use change, run a simulation and view the results in the form of the future land use map and a selected set of indicators. All data is stores on the servers of the European Commissions Joint Research Centre.  

Note that MOLAND-Light is a simplified model and therefore  only intended for demonstration purposes, not for research or policy analysis.
